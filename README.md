**PeopleFlow** is a production-grade **Human Resource Management (HRM)** and **Enterprise Resource Planning (ERP)** dashboard designed for modern workforces. Engineered with a scalable, **feature-based architecture**, it provides a seamless user experience for both employees and administrators.

---

## 🚀 Key Modules & Capabilities

### 📊 **Intelligent Dashboard**
- **Data Visualization**: Real-time analytics of attendance trends, leave balances, and upcoming holidays using `Recharts`.
- **Global Overview**: Unified view for management to monitor organizational health at a glance.

### ⏱️ **Smart Attendance Tracking**
- **Precision Logging**: Automated check-in/out logic with location and time synchronization.
- **Monthly Insights**: Detailed attendance summaries and easy correction request workflows.
- **Admin Oversight**: Centralized monitoring of employee work hours and punctuality.

### 🌴 **Seamless Leave Management**
- **Full Lifecycle**: Apply, track, and manage leave requests (Approve/Reject) with instant status notifications.
- **Policy Enforcement**: Automated balance tracking and holiday integration.

### 👥 **Employee Lifecycle Management**
- **Effortless Onboarding**: Streamlined CRUD operations for managing the entire employee journey.
- **Role-Based Profiles**: Personalized dashboards for `Admin`, `Manager`, and `Employee` roles.

### 🔐 **Enterprise-Grade Security**
- **Role-Based Access Control (RBAC)**: Robust permission system ensuring data isolation.
- **Secure Authentication**: Stateless JWT-based auth flow with protected route wrappers.

---

## 🏗️ Technical Excellence (The Foundation)

This project isn't just about features; it's about **clean, maintainable code**:

- **📁 Vertical Slice Architecture**: Logic, state, and UI are encapsulated within feature modules (e.g., `features/attendance`), making the system infinitely scalable.
- **⚡ Performance Optimized**: Leveraging **TanStack Query** for advanced server-state management (caching, background syncing, and optimistic updates).
- **🎨 Modern UI/UX**: Built with **Shadcn UI** and **Radix UI** primitives for a premium, accessible, and responsive design system.
- **🛡️ Type-Safety**: 100% **TypeScript** coverage, from API services to UI components, ensuring a bug-resistant codebase.

## ⚡ Quick Start

### 1. Installation
```bash
git clone https://github.com/RakibHassan11/hr-management-system.git
npm install
```

### 2. Configure Environment
Create a `.env` file in the root:
```env
VITE_API_URL=https://your-api.com/api
```

### 3. Run Locally
```bash
npm run dev
```
